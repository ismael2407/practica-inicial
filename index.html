5.¿Qué es y que función cumple el CSS?

CSS, en español «Hojas de estilo en cascada», es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado.

Es un lenguaje informático que especifica cómo se presentan los documentos a los usuarios: cómo se diseñan, compaginan, etc.

6.¿como se incorpora CSS con HTML?

Se puede incluir CSS en HTML:

a.Vinculando a un archivo de hoja de estilo separado.

b.Incrustando CSS en una etiqueta de estilo.

c.Agregando estilos en línea a elementos HTML.

d.Importando archivos de hojas de estilo existentes.

e.Inyectando css con javascript.

7.¿Cuál es la anatomía de una regla CSS?

Una regla CSS siempre tiene un selector (el elemento al que quiero modificar) y una declaración (las características que le quiero agregar o cambiar). La declaración va encerrada entre llaves { } y dentro de ella se escriben todas las propiedades con sus valores, que modificarán al selector.

8.¿Qué es javascript? ¿Podrías definirlo como un lenguaje de programación? ¿De qué tipo? Justificar.

JavaScript es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

se puede considerar un lenguaje de programacion ya que le proporcionan a una persona, la capacidad de escribir una serie de instrucciones o secuencias de ordenes en forma de algorintmos con el fin de controlar el conportamiento fisico o logico de un sistema informatico, lo cual es exactamente lo que hace javascript al permitirle crear a una persona, por ejemplo, una pagina web
